# Othello

A Python implementation of the Othello game with GUI and AI players.

## Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the game:
```bash
python main.py
```

## Structure

- `src/othello/core`: Core game logic (Board, Game).
- `src/othello/gui`: GUI implementation using Tkinter.
- `src/othello/player`: AI players (Computer, CNN, MCS).
- `src/othello/utils`: Utility scripts (Wthor parser, Scraping).
- `models`: Trained models.
- `tests`: Unit tests.
